# 🇺🇸 English

## 📖 About the Project

**Vale do Açaí** is a fictional city website created during my learning period as a **Young Apprentice** at **Via de Acesso**.

The main objective of this project was to practice front-end web development fundamentals while designing a complete and visually engaging website around an original fictional city.

The website introduces visitors to the culture, economy, tourism, and geography of **Vale do Açaí**, demonstrating responsive layouts, interactive elements, animations, and clean interface design using only HTML, CSS, and JavaScript.

---

## ✨ Features

### 🏞 Modern Landing Page
- Clean and immersive hero section
- Responsive navigation menu
- Smooth visual presentation

### 🎠 Interactive Tourist Attractions Carousel
- Navigation buttons
- Pagination indicators
- Touch swipe support for mobile devices
- Automatic slideshow
- Individual pages for every tourist attraction

### 👥 Animated Population Counter
- Scroll-triggered animation
- Dynamic counting effect
- Simulated city population display

### 📊 Economy Dashboard
- Animated bar chart
- Fictional açaí production statistics
- Visual representation of annual growth

### 🗺 Territory Section
- Placeholder prepared for a future interactive city map

### 📱 Responsive Design
- Desktop support
- Tablet support
- Mobile support

### 🎨 Custom UI
- Custom animations
- Modern typography
- Green color palette inspired by the Amazon rainforest
- Smooth transitions and hover effects

# 🇧🇷 Português

## 📖 Sobre o Projeto

**Vale do Açaí** é um website de uma cidade fictícia desenvolvido durante meu período de aprendizado como **Jovem Aprendiz** na instituição **Via de Acesso**.

O objetivo do projeto foi colocar em prática os fundamentos do desenvolvimento web front-end por meio da criação de um site completo para uma cidade fictícia, explorando conceitos de design, responsividade e interatividade utilizando apenas tecnologias nativas da web.

O site apresenta a cultura, economia, turismo e território da cidade fictícia **Vale do Açaí**, oferecendo uma experiência visual moderna e agradável.
